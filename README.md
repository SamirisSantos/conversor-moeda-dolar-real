# Conversor de Moedas em ReactJS

> Aplicação para converter moeda Dólar em Real com taxa de imposto do estado USA e a taxa IOF

>Conxeção com a [API](https://docs.awesomeapi.com.br/api-de-moedas) para consultar o dolar em tempo real

# iniciar o projeto

execute 'npm start' e acesse localmente por http://localhost:3000/

# instalar estilo 

Estilo utilizado reactstrap (https://reactstrap.github.io/)

Instale o reactstrap e o Bootstrap do NPM.

npm install --save bootstrap
npm install --save reactstrap react react-dom

Importe o CSS do Bootstrap no src/index.jsarquivo:

import 'bootstrap/dist/css/bootstrap.min.css';

Importe os componentes reatstrap necessários no src/App.js ou nos arquivos de componentes personalizados:

import { Button } from 'reactstrap';
